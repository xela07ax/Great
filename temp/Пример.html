Это пример разметки текста для публикации на бирже статей WebArtex. Текст должен быть интересным и читабельным. Не должен содержать стилистических и грамматических ошибок.


HTML тэги, разрешенные для использования:
Для разметки текстов разрешено использовать следующие тэги:

    Заголовки второго или третьего уровня (, )
    Текстовый абзац ()
    Нумерованные и ненумерованные списки (, , )
    Тэги для создания ссылки ()


Использование тэгов, которые не указаны в списке, запрещено. Такие тэги будут автоматически удаляться системой.
Ссылки из текста:
В тексте возможна простановка одной, или двух ссылок. Ссылки в обязательном порядке должны вести на один домен. Ссылки не должны содержать ключевых слов, которые относятся к запрещенным для продвижения тематикам.
